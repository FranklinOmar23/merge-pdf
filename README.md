# Merge PDF

Aplicación web hecha con React, TypeScript y Vite para trabajar con archivos PDF desde el navegador.

## Funciones

- Unir varios archivos PDF en un solo documento.
- Separar uno o varios PDF en páginas individuales.
- Convertir imágenes a un archivo PDF.
- Reordenar archivos por arrastre antes de procesarlos.

## Desarrollo

```bash
npm install
npm run dev
```

## Producción

```bash
npm run build
npm run preview
```

## Notas

- El proyecto usa pdf-lib para procesar documentos en el navegador.
- Los estilos utilitarios se cargan mediante Tailwind CSS por CDN en [index.html](index.html).
